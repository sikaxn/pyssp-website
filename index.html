Python SSP (pySSP)

An open source alternative to Sports Sounds Pro for show playback and arena production on Windows and macOS Apple silicon.

  • Download for Windows and Mac
  • Documentation

Built to replace the old workflow

Python SSP keeps the familiar Sports Sounds Pro workflow while adding modern control options and an open codebase. It supports Windows and macOS on Apple silicon, and is aimed at operators who want the core SSP feature set without being locked to the old software.

Features

Classic SSP Features

Most of the features operators expect from older SSP systems are already available in Python SSP.

Timecode Output

Drive external systems with timecode output for tighter playback sync during shows.

Web Remote

Control playback remotely from a browser when you need fast access away from the main machine.

MIDI Control

Map playback actions to MIDI hardware for hands-on operation in live production setups.

pySSP

An open source, free alternative to older SSP soundboard software, with more functionality for modern show playback workflows.

  • Download

Software focused on real show workflows

pySSP helps teams reduce repetitive prep tasks, validate show data, and keep playback workflows predictable under production pressure. It is intended for practical use in rehearsal and performance pipelines where reliability matters more than novelty.

Core Capabilities

Automation Scripts

Build repeatable scripts for setup, validation, and maintenance across show projects.

Cue Management

Support cue edits and cue-point updates with less manual work and fewer hand-entry errors.

Large Project Navigation

Speed up lookups and bulk operations for bigger libraries and complex productions.

Timecode Workflow Support

Strengthen consistency for timecode-driven playback preparation and verification.

Page and Group Actions

Handle page-level and group-level updates in bulk when projects scale up.

Living Documentation

Published docs are continuously built to keep setup steps and API references current.

Interface Coverage

Typical Workflow

  1. Prepare or load a show project and inspect current cue/page structure.
  2. Run pySSP scripts to apply repetitive updates and data checks.
  3. Validate playback behavior in UI areas such as timecode, DSP, and routing.
  4. Publish or share outputs with confidence before rehearsal or live use.
